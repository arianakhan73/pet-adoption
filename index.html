Multivariate Regression notebook

We used this notebook to create our graph for our prediction

Import sources (we already did that above)

We chose the y variable output (all other variables are used as input), as well as the test_size, random_state, and alpha

In[44]:
y_column = "age_years"

test_size = 0.2
random_state = 1234
alpha = 0.01

The data is divided into training and test sets:

In[45]:
X = df.drop(y_column, axis=1)
X = pd.get_dummies(X)
y = df[[y_column]]
X_train, X_test, y_train, y_test = model_selection.train_test_split(X, y, test_size = test_size, random_state=random_state)
In[46]:
reg = linear_model.Lasso(alpha = alpha).fit(X_train, y_train)
In[47]:
y_pred = reg.predict(X_test)

Training and test scores are printed (close to 0 is bad, close to 1 is good).

In[48]:
print("training score: {:.3f}".format(reg.score(X_train, y_train)))
